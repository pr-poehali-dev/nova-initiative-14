{НАЗВАНИЕ} — сервис наставничества для студентов УрФУ (направление «Конструирование», машиностроение), где практикующие инженеры‑конструкторы помогают довести дипломный проект до осознанной защиты. **Шаг 4 — Проверка и правки**
Вы сдаёте главы/чертежи → наставник проверяет логику, структуру, оформление → вы получаете комментарии и план правок → дорабатываете.

#### Модуль 1: Тема и задание на проектирование

**Результат:** Чётко сформулированная тема, заполненное задание на ВКР, понимание объекта и предмета проектирования.

**Что делаем:**
- Анализируем формулировку темы — достаточно ли она конкретна
- Помогаем уточнить цель и задачи проектирования
- Разбираем требования задания на ВКР (что именно от вас ждут)
- Составляем календарный план работы

**Типовые ошибки студентов:**
- Расплывчатая формулировка темы, которая не ведёт к конкретному результату
- Путают цель и задачи, объект и предмет
- Не читают задание на ВКР внимательно — пропускают обязательные разделы

**Артефакты:** заполненное задание на ВКР, план работы с дедлайнами, формулировки цели/задач.

---

#### Модуль 2: Обзор литературы и аналитическая часть

**Результат:** Структурированный обзор с выводами, обосновывающими ваше проектное решение.

**Что делаем:**

**Типовые ошибки студентов:**

- Нет логической связи между обзором и проектным решением
- Использование устаревших или нерелевантных источников

**Артефакты:** готовая глава обзора (написанная студентом, проверенная наставником), список источников.

- (A) Больше 3 месяцев → +3мес
- (B) 1–3 месяца → +инд1м, +групп1м
- (C) 2–4 недели → +инд1м
- (D) Меньше недели → +экспресс

**Q2: Работа уже начата?**
- (A) Нет, ещё не начинал → +3мес
- (B) Есть черновик/часть глав → +инд1м, +групп1м
- (C) Почти готова, но замечания → +инд1м, +экспресс
- (D) Готова, но не понимаю материал → +экспресс

**Q3: Есть ли замечания научного руководителя?**
- (A) Нет → нейтрально
- (B) Есть, немного → +инд1м
- (C) Много, не понимаю что исправлять → +инд1м, +экспресс
- (D) Не допускают к защите → +экспресс

**Q4: Готовы работать в группе?**
- (A) Да, мне проще в группе → +групп1м
- (B) Нет, нужен индивидуальный подход → +инд1м, +3мес
- (C) Не знаю → нейтрально

**Q5: Какой бюджет?**
- (A) Минимальный → +групп1м, +экспресс
- (B) Средний → +инд1м
- (C) Готов вложиться в качество → +3мес

**Q6: Что для вас сейчас важнее всего?**
- (A) Понять материал и сделать качественно → +3мес
- (B) Закрыть замечания и подготовиться к защите → +инд1м
- (C) Получить поддержку и структуру → +групп1м
- (D) Срочно разобраться перед защитой → +экспресс

**Логика подсчёта:** Считаем баллы для каждого тарифа. Тариф с максимальным количеством «+» = рекомендация. При равенстве — показываем оба.
