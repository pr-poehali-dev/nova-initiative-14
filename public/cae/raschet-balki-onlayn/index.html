Расчёт балки онлайн

Бесплатный онлайн-калькулятор балки считает прочность и прогиб прямо в браузере. Задайте опоры, нагрузки и сечение — получите эпюры и проверку по нормам.

Что вы получите

  • Эпюра Q — распределение поперечных сил по длине балки.
  • Эпюра M — изгибающие моменты с поиском опасного сечения.
  • Прогиб — максимальный прогиб и сравнение с допустимым по ГОСТ.
  • Напряжения по Мизесу — оценка прочности материала.
  • Подбор сечения — выбор профиля из каталога ГОСТ под вашу нагрузку.

Удобно для дипломного проекта: результат можно выгрузить в PDF-отчёт по ЕСКД и вставить в пояснительную записку. Для коммерческого сайта есть встраиваемый виджет калькулятора балки.
